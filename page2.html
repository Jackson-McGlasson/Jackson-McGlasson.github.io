No Balls to the Walls

This project is a game made using p5.js. It uses a drawing program that creates sprites and shapes on screen. The aim of the game is to stay alive as long as possible by avoiding having your square collide with the walls. You can change direction with WASD to make it move up, left, down, and right. I choose this idea because it is a simple yet, entertaining concept for a game. For this project I liked working on how to factor in the velocities of the squares into a proper gameplay.

Controls:
W- Move Up
A- Move Left
S- Move Down
D- Move Right
Enter- Start
R- Reset
